O projekcie

Projekt to interaktywna tabela HTML prezentująca 13 nazwanych kolorów CSS — od klasycznych (Red, Blue, Green) po mniej oczywiste (Teal, Maroon, Fuchsia). Każda komórka pokazuje kolor jako tło, jego angielską i polską nazwę, kod HEX oraz wartości RGB z linkiem do Wikipedii.

Projekt demonstruje zaawansowane łączenie komórek tabelarycznych przez colspan (scalanie kolumn) i rowspan (scalanie wierszy), a ostatnia komórka to w pełni interaktywny color picker — wybierasz kolor, a komórka zmienia tło i wyświetla jego HEX i RGB w czasie rzeczywistym.

Struktura tabeli — colspan i rowspan

Tabela ma 6 kolumn bazowych. Komórki są scalane nieregularnie — Red zajmuje 2×3 (colspan=2, rowspan=3), Gold zajmuje 1×2 (rowspan=2), a Teal i Pink rozciągają się na całą szerokość. Poniżej schemat układu:

Kluczowe techniki — przykłady kodu

1 Scalanie komórek — colspan i rowspan 

<!-- Red zajmuje 2 kolumny i 3 wiersze -->
<td bgcolor="Red"
     colspan="2"
     rowspan="3"
     height="200" width="200">
  Red / czerwony / #FF0000
</td>

<!-- Teal rozciąga się na całą szerokość (6 kolumn) -->
<td bgcolor="Teal"
     colspan="6"
     height="200">
  Teal / morski / #008080
</td>

2 Interaktywny color picker — JavaScript 

<!-- HTML: komórka z inputem -->
<td id="customCell" colspan="6">
  <label for="colorPicker">Choose color</label>
  <input type="color" id="colorPicker" value="#ffffff">
  <span id="hexValue">#??????</span>
  <span id="rgbValue">RGB (???, ???, ???)</span>
</td>

// JavaScript — konwersja HEX → RGB i aktualizacja komórki
function hexToRgb(hex) {
  let r = parseInt(hex.substr(1, 2), 16);
  let g = parseInt(hex.substr(3, 2), 16);
  let b = parseInt(hex.substr(5, 2), 16);
  return `(${r}, ${g}, ${b})`;
}

document.getElementById('colorPicker')
  .addEventListener('input', function() {
    const hex = this.value;
    document.getElementById('customCell').style.backgroundColor = hex;
    document.getElementById('hexValue').textContent = hex;
    document.getElementById('rgbValue').textContent = "RGB " + hexToRgb(hex);
  });

3 Animowany gradient RGB na tekście 

/* Gradient tęczowy na tekście — kluczowe właściwości */
.rgb-title {
  background: linear-gradient(
    270deg, red, orange, yellow,
    green, cyan, blue, violet, red
  );
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip:         text;
  background-size:         1000% 100%; /* duży, żeby animacja była płynna */
  animation: rgbText 10s linear infinite;
}

@keyframes rgbText {
  0%   { background-position: 0%   50%; }
  100% { background-position: 100% 50%; }
}
