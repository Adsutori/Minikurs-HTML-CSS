Czym jest DOM?

DOM — Document Object Model — to reprezentacja dokumentu HTML w postaci drzewa obiektów. Kiedy przeglądarka wczytuje stronę, nie widzi surowego tekstu HTML — buduje z niego strukturę, po której JavaScript może się poruszać i którą może modyfikować.

Jak wygląda drzewo DOM?

Dla takiego HTML:

<html>
  <head>
    <title>Strona</title>
  </head>
  <body>
    <h1>Witaj!</h1>
    <p>To jest <strong>ważny</strong> tekst.</p>
  </body>
</html>

Przeglądarka buduje takie drzewo:

Document
└── html
    ├── head
    │   └── title
    │       └── "Strona"
    └── body
        ├── h1
        │   └── "Witaj!"
        └── p
            ├── "To jest "
            ├── strong
            │   └── "ważny"
            └── " tekst."

Dlaczego DOM jest ważny?

Dzięki DOM JavaScript może dynamicznie zmieniać stronę bez jej przeładowania: 

// Zmień tekst nagłówka
document.querySelector('h1').textContent = 'Nowy tytuł';

// Dodaj klasę do elementu
document.querySelector('.karta').classList.add('aktywna');

// Utwórz nowy element i dodaj go do strony
const nowy = document.createElement('p');
nowy.textContent = 'Nowy akapit!';
document.body.appendChild(nowy);

Inspektor DOM w przeglądarce

Każda przeglądarka ma wbudowane narzędzia deweloperskie (F12), gdzie możesz podglądać i edytować DOM na żywo:

  • Elements (Chrome/Edge) / Inspector (Firefox) — drzewo DOM
  • Kliknij prawym przyciskiem na element → Zbadaj element
  • Możesz edytować HTML i CSS bezpośrednio w przeglądarce (zmiany są tymczasowe)
💡 DOM to interfejs między HTML a JavaScript. Dobra znajomość struktury DOM jest kluczowa gdy zaczniesz uczyć się JavaScript — wszystkie operacje na stronie (animacje, dynamiczne treści, formularze) przechodzą właśnie przez DOM.
