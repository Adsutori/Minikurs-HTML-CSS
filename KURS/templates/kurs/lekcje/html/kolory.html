Kolory w HTML

Kolory w HTML możesz definiować na kilka sposobów. Używa się ich w atrybutach stylów inline oraz w CSS. Warto znać wszystkie formaty, bo spotkasz je w każdym projekcie webowym.

Nazwy kolorów

HTML rozpoznaje ponad 140 nazwanych kolorów. Oto kilka przykładów: 

<p style="color: red">Czerwony</p>
<p style="color: navy">Granatowy</p>
<p style="color: forestgreen">Leśna zieleń</p>
<p style="color: tomato">Pomidorowy</p>

Zapis szesnastkowy (HEX)

Najczęściej stosowany format. Zapis #RRGGBB — dwie cyfry hex dla każdego kanału: czerwonego, zielonego i niebieskiego. Wartości od 00 (brak) do FF (maksimum). 

#FF0000  → czerwony
#00FF00  → zielony
#0000FF  → niebieski
#FFFFFF  → biały
#000000  → czarny
#808080  → szary
#FF5733  → koralowy

Jeśli obie cyfry w każdej parze są takie same, można skrócić zapis do 3 znaków:

#FF0000 → #F00
#FFFFFF → #FFF
#000000 → #000

Format RGB

Trzy liczby od 0 do 255 dla kanałów czerwonego, zielonego i niebieskiego: 

style="color: rgb(255, 0, 0)"      → czerwony
style="color: rgb(0, 128, 0)"      → zielony
style="color: rgb(100, 100, 100)"  → szary

Format RGBA — z przezroczystością

Czwarty parametr (alfa) określa przezroczystość: 0 to pełna przezroczystość, 1 to pełna nieprzezroczystość. 

style="background-color: rgba(255, 0, 0, 0.3)"  → półprzezroczysty czerwony

Format HSL

Odcień (0–360°), nasycenie (0–100%) i jasność (0–100%). Intuicyjny dla projektantów: 

style="color: hsl(0, 100%, 50%)"    → czerwony
style="color: hsl(120, 100%, 50%)"  → zielony
style="color: hsl(240, 100%, 50%)"  → niebieski

Kolor tła i tekstu

<p style="color: white; background-color: #333">
  Biały tekst na ciemnym tle
</p>
