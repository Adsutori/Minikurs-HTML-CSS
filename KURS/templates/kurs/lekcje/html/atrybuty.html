Atrybuty globalne

Atrybuty globalne to takie, które można dodać do dowolnego znacznika HTML. Niezależnie od tego czy to <p>, <div> czy <button> — te atrybuty zawsze działają.

id — unikalny identyfikator

<h2 id="sekcja-kontakt">Kontakt</h2>

<!-- Odwołanie w CSS -->
#sekcja-kontakt { color: navy; }

<!-- Odwołanie w HTML (kotwica) -->
<a href="#sekcja-kontakt">Przejdź do kontaktu</a>

Wartość id musi być unikalna na całej stronie — nie może powtórzyć się dwa razy w jednym dokumencie.

class — klasa stylu

<p class="wazny">Ten akapit jest ważny.</p>
<p class="wazny duzy">Ważny i duży — dwie klasy.</p>

<!-- Odwołanie w CSS -->
.wazny { font-weight: bold; }
.duzy  { font-size: 1.5rem; }

W odróżnieniu od id, klasa może być użyta wielokrotnie. Element może mieć wiele klas — oddzielamy je spacją.

style — styl inline

<p style="color: red; font-size: 18px; margin-top: 20px">
  Czerwony tekst 18px
</p>

Style inline mają najwyższy priorytet w CSS — nadpisują style z arkuszy. Używaj ich oszczędnie, tylko gdy naprawdę potrzebujesz nadpisać styl w jednym miejscu.

title — dymek z opisem

<abbr title="HyperText Markup Language">HTML</abbr>
<img src="foto.jpg" alt="Zdjęcie" title="Kliknij, aby powiększyć">
<button title="Usuń ten element">🗑️</button>

Tekst z title pojawia się jako dymek po najechaniu kursorem.

lang — język elementu

<p lang="en">This paragraph is in English.</p>
<p lang="de">Dieser Absatz ist auf Deutsch.</p>

Przydatne gdy na polskiej stronie cytujesz tekst w innym języku — czytnik ekranowy przełączy odpowiedni głos.

tabindex — kolejność fokusa

<input type="text" tabindex="1">
<input type="text" tabindex="3">
<input type="text" tabindex="2">

Określa kolejność przechodzenia przez elementy klawiszem Tab. tabindex="0" — naturalna kolejność DOM. tabindex="-1" — element pominięty przy nawigacji klawiaturą.

hidden — ukrycie elementu

<div hidden>Ten element jest niewidoczny.</div>

data-* — własne atrybuty danych

<button data-user-id="42" data-action="usun">Usuń użytkownika</button>
<div data-kategoria="html" data-poziom="podstawowy">...</div>

Atrybuty data-* pozwalają przechowywać dowolne dane w elementach HTML. Są niewidoczne dla użytkownika, ale dostępne dla JavaScript przez element.dataset.userId.

contenteditable — edytowalny element

<p contenteditable="true">
  Kliknij tutaj i edytuj ten tekst bezpośrednio w przeglądarce.
</p>

draggable — przeciąganie

<div draggable="true">Przeciągnij mnie!</div>
💡 Zasada: id używaj do unikalnych elementów (kotwice, JS, formularze). class używaj do stylowania i grupowania podobnych elementów. Nigdy nie używaj id tylko po to, żeby nadać styl — od tego jest class.
