Inputy

Element <input> to najbardziej wszechstronny znacznik formularzy HTML. Dzięki atrybutowi type może przyjmować dziesiątki różnych form — od pola tekstowego po suwak kolorów.

Typy inputów tekstowych

<input type="text" placeholder="Imię i nazwisko">
<input type="email" placeholder="adres@email.com">
<input type="password" placeholder="Hasło">
<input type="search" placeholder="Szukaj...">
<input type="url" placeholder="https://example.com">
<input type="tel" placeholder="+48 501 234 567">

Typy inputów numerycznych i zakresowych

<input type="number" min="0" max="100" step="1">
<input type="range" min="0" max="100" value="50">

Typy inputów daty i czasu

<input type="date">
<input type="time">
<input type="datetime-local">
<input type="month">
<input type="week">

Inne typy

<input type="color" value="#ff0000">
<input type="file" accept=".pdf,.jpg,.png">
<input type="checkbox"> Akceptuję regulamin
<input type="radio" name="plec" value="m"> Mężczyzna
<input type="radio" name="plec" value="k"> Kobieta

Przyciski

<input type="submit" value="Wyślij">
<input type="reset" value="Wyczyść">
<input type="button" value="Kliknij mnie">
<input type="hidden" name="token" value="abc123">

Ważne atrybuty inputów

AtrybutOpis
nameNazwa pola — klucz w danych formularza
valueDomyślna wartość pola
placeholderPodpowiedź wyświetlana w pustym polu
requiredPole obowiązkowe — formularz nie wyśle się bez niego
disabledPole nieaktywne (szare, nieklikalnie)
readonlyPole tylko do odczytu
maxlengthMaksymalna liczba znaków
min / maxZakres dla pól numerycznych i dat
patternWyrażenie regularne do walidacji
autocompleteon / off — podpowiedzi przeglądarki
💡 Każdy <input> powinien mieć powiązany <label>. Bez niego użytkownik nie wie co wpisać, a czytniki ekranowe nie mogą opisać pola. Powiązanie przez atrybut for = id inputu.
