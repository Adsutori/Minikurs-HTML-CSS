Wstęp do HTML

HTML, czyli HyperText Markup Language, to język znaczników używany do tworzenia stron internetowych. Nie jest to język programowania — nie zawiera logiki ani obliczeń. Jego zadaniem jest opisanie struktury i treści dokumentu: co jest nagłówkiem, co akapitem, co obrazkiem, a co odnośnikiem.

Każda strona internetowa, którą widzisz w przeglądarce, zbudowana jest z HTML-a. CSS odpowiada za wygląd, JavaScript za interaktywność — ale fundamentem zawsze jest HTML.

Krótka historia

HTML powstał na początku lat 90. XX wieku. Jego twórcą jest Tim Berners-Lee, który pracując w CERN, potrzebował sposobu na łatwe udostępnianie dokumentów naukowych przez sieć. Dziś HTML jest standardem utrzymywanym przez organizację W3C (World Wide Web Consortium).

Aktualna wersja to HTML5, która wprowadziła wiele nowych możliwości: obsługę wideo i audio, nowe znaczniki semantyczne oraz wsparcie dla aplikacji webowych.

Jak działa przeglądarka?

Kiedy wpisujesz adres strony w przeglądarce, ona pobiera plik HTML z serwera i renderuje go — czyli interpretuje znaczniki i wyświetla gotową stronę. Przeglądarka czyta kod od góry do dołu, budując strukturę zwaną DOM (Document Object Model).

Co będziesz umiał po tym kursie?

  • Tworzyć poprawnie zbudowane strony HTML5
  • Używać znaczników do formatowania tekstu, list, tabel i formularzy
  • Wstawiać obrazy, wideo i audio
  • Tworzyć hiperłącza i nawigację
  • Rozumieć semantykę i dostępność stron
  • Zbudować własne CV i prostą stronę WWW
💡 Wskazówka: Do pisania HTML wystarczy zwykły edytor tekstu (np. VS Code, Notepad++). Nie potrzebujesz żadnego specjalnego oprogramowania — tylko przeglądarki do podglądu efektów.
