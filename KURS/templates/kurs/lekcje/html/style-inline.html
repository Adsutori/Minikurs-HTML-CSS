Style inline

Style inline to style CSS zapisane bezpośrednio w atrybucie style konkretnego elementu HTML. Działają natychmiast i mają najwyższy priorytet w kaskadzie CSS — ale właśnie dlatego należy używać ich z rozwagą.

Składnia

<p style="color: red; font-size: 18px; margin-top: 20px">
  Czerwony tekst, rozmiar 18px, margines górny 20px
</p>

Wartość atrybutu style to lista właściwości CSS oddzielonych średnikami. Każda właściwość ma format właściwość: wartość.

Priorytety CSS — kaskada

CSS stosuje ścisłą hierarchię ważności. Od najniższego do najwyższego priorytetu:

  1. Style przeglądarki (domyślne)
  2. Zewnętrzny arkusz CSS (<link rel="stylesheet">)
  3. Style wewnętrzne (<style> w <head>)
  4. Style inline — nadpisują wszystko powyżej
  5. !important — nadpisuje nawet style inline (używaj tylko w ostateczności)

Przykład konfliktu priorytetów

<!-- W pliku style.css -->
p { color: blue; }

<!-- W <head> -->
<style>
  p { color: green; }
</style>

<!-- W HTML -->
<p style="color: red">Jaki kolor?</p>
<!-- Wynik: czerwony — style inline wygrywają -->

Kiedy używać stylów inline?

SytuacjaInline?
Dynamiczne style generowane przez JavaScript✅ Tak
Jednorazowy wyjątek od reguły CSS⚠️ Ostrożnie
Szybki test podczas developmentu✅ Tymczasowo
Szablony e-mail HTML✅ Tak — klienty pocztowe często ignorują zewnętrzny CSS
Stylowanie całej strony lub wielu elementów❌ Nie — użyj arkusza CSS
Nadpisywanie stylów komponentów❌ Nie — użyj klas CSS

Dlaczego nie nadużywać?

  • Trudne w utrzymaniu — zmiana koloru na 50 elementach wymaga edycji 50 plików
  • Brak możliwości reużycia — nie możesz zastosować tego samego stylu do wielu elementów jedną regułą
  • Brak pseudoklas — nie możesz napisać style="color:red :hover"
  • Zaśmiecony HTML — kod staje się nieczytelny

Alternatywa — klasy CSS

<!-- ❌ Źle — powtarzasz ten sam styl inline -->
<p style="color: red; font-weight: bold">Błąd 1</p>
<p style="color: red; font-weight: bold">Błąd 2</p>
<p style="color: red; font-weight: bold">Błąd 3</p>

<!-- ✅ Dobrze — jedna klasa, wiele zastosowań -->
<style>
  .blad { color: red; font-weight: bold; }
</style>

<p class="blad">Błąd 1</p>
<p class="blad">Błąd 2</p>
<p class="blad">Błąd 3</p>
💡 Zasada: Jeśli ten sam styl inline pojawia się więcej niż raz — to sygnał, że powinna być klasa CSS. Style inline są jak wyjątki od reguły, nie jak reguła sama w sobie.
