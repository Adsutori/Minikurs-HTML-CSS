ID i Class

Atrybuty id i class to podstawowe narzędzia do identyfikowania i grupowania elementów HTML. Bez nich CSS i JavaScript nie wiedziałyby, do których elementów się odwoływać.

ID — unikalny identyfikator

<header id="glowny-naglowek">...</header>
<section id="o-nas">...</section>
<footer id="stopka">...</footer>

Zasady używania id:

  • Musi być unikalne na całej stronie
  • Nie może zawierać spacji
  • Może zawierać litery, cyfry, myślniki i podkreślenia
  • Powinno zaczynać się od litery

Class — wielokrotna klasa

<p class="tekst-wazny">Ważna informacja.</p>
<p class="tekst-wazny podswietlony">Ważna i podświetlona.</p>
<div class="karta aktywna">Aktywna karta.</div>

Klasy można dowolnie łączyć — oddzielasz je spacją. Ta sama klasa może być użyta na wielu elementach.

Odwołania w CSS

/* Selektor ID — poprzedzony # */
#glowny-naglowek {
  background-color: #1a1a2e;
  color: white;
}

/* Selektor klasy — poprzedzony . */
.tekst-wazny {
  font-weight: bold;
  color: #c0392b;
}

/* Element z konkretną klasą */
p.podswietlony {
  background-color: yellow;
}

Odwołania w JavaScript

// Pobierz element po ID
const naglowek = document.getElementById('glowny-naglowek');

// Pobierz pierwszy element o klasie
const wazny = document.querySelector('.tekst-wazny');

// Pobierz wszystkie elementy o klasie
const wszystkieWazne = document.querySelectorAll('.tekst-wazny');
