Typografia w CSS

Typografia to sztuka doboru i układu tekstu. W CSS masz pełną kontrolę nad czcionkami, rozmiarem, odstępami i wyglądem tekstu — od prostych blogów po zaawansowane systemy projektowe.

font-family — rodzina czcionek

/* Lista zapasowa — przeglądarka użyje pierwszej dostępnej */
body {
  font-family: 'Inter', -apple-system, BlinkMacSystemFont,
               'Segoe UI', Roboto, sans-serif;
}

/* Kategorie generyczne */
font-family: serif;       /* Times New Roman, Georgia */
font-family: sans-serif;  /* Arial, Helvetica, Inter */
font-family: monospace;   /* Courier, JetBrains Mono */
font-family: cursive;     /* pismo odręczne */
font-family: fantasy;     /* dekoracyjne */

Google Fonts — zewnętrzne czcionki

<!-- W <head> HTML -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap"
      rel="stylesheet">

/* W CSS */
body {
  font-family: 'Inter', sans-serif;
}

font-size — rozmiar czcionki

/* Piksele — absolutne */
font-size: 16px;

/* rem — względem root (html) — zalecane */
font-size: 1rem;    /* = 16px jeśli html = 16px */
font-size: 1.5rem;  /* = 24px */
font-size: 0.875rem; /* = 14px */

/* em — względem rodzica */
font-size: 1.2em;   /* 120% rozmiaru rodzica */

/* % — względem rodzica */
font-size: 120%;

/* Responsywny rozmiar */
font-size: clamp(1rem, 2.5vw, 2rem); /* min | preferowany | max */

font-weight — grubość

font-weight: 100;  /* Thin */
font-weight: 300;  /* Light */
font-weight: 400;  /* Regular (normal) */
font-weight: 500;  /* Medium */
font-weight: 600;  /* SemiBold */
font-weight: 700;  /* Bold */
font-weight: 800;  /* ExtraBold */
font-weight: 900;  /* Black */

/* Słowa kluczowe */
font-weight: normal;  /* = 400 */
font-weight: bold;    /* = 700 */

line-height — wysokość linii

/* Liczba bez jednostki — zalecane (mnożnik font-size) */
line-height: 1;     /* ścisłe */
line-height: 1.5;   /* komfortowe dla tekstu */
line-height: 1.8;   /* luźne, czytelne */

/* Z jednostką */
line-height: 24px;
line-height: 150%;

letter-spacing i word-spacing

/* Odstęp między literami */
letter-spacing: 0.05em;   /* lekko rozstrzelony */
letter-spacing: 0.1em;    /* nagłówki, przyciski */
letter-spacing: -0.02em;  /* ściśnięty (duże nagłówki) */

/* Odstęp między słowami */
word-spacing: 4px;

text-align — wyrównanie

text-align: left;     /* domyślne */
text-align: center;
text-align: right;
text-align: justify;  /* wyjustowany — ostrożnie na małych ekranach */

text-decoration

text-decoration: none;           /* usuwa podkreślenie z linków */
text-decoration: underline;
text-decoration: line-through;   /* przekreślenie */
text-decoration: overline;

/* Szczegółowa kontrola */
text-decoration-color: red;
text-decoration-style: dashed;
text-decoration-thickness: 2px;
text-underline-offset: 4px;      /* odstęp podkreślenia od tekstu */

text-transform

text-transform: uppercase;    /* WIELKIE LITERY */
text-transform: lowercase;    /* małe litery */
text-transform: capitalize;   /* Pierwsza Litera Wielka */
text-transform: none;

Gradient na tekście

.gradient-text {
  background: linear-gradient(135deg, #7c3aed, #a855f7);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}
💡 Zasada rem vs em: Używaj rem do rozmiarów czcionek (zawsze relatywne do root, przewidywalne) i em do paddingów/marginesów komponentów (skalują się razem z czcionką komponentu).
