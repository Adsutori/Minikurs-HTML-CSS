Zaawansowane przejścia i mikro-animacje

Dobrze zaprojektowane przejścia to różnica między stroną, która "działa" a stroną, która "żyje". Mikro-animacje komunikują stan, kierują uwagę i budują zaufanie użytkownika.

Krzywe Béziera — własny easing

/* cubic-bezier(x1, y1, x2, y2) */

/* Sprężysty — overshoots */
transition: transform 0.4s cubic-bezier(0.34, 1.56, 0.64, 1);

/* Material Design standard */
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);

/* Szybki start, wolne zatrzymanie */
transition: transform 0.5s cubic-bezier(0.16, 1, 0.3, 1);

/* Dramatyczne wejście */
transition: opacity 0.6s cubic-bezier(0.22, 1, 0.36, 1);

/* Własne zmienne dla spójności */
:root {
  --ease-spring:   cubic-bezier(0.34, 1.56, 0.64, 1);
  --ease-smooth:   cubic-bezier(0.4, 0, 0.2, 1);
  --ease-snappy:   cubic-bezier(0.16, 1, 0.3, 1);
  --ease-bounce:   cubic-bezier(0.68, -0.55, 0.265, 1.55);
}

Przejścia stanów UI

/* Przycisk z efektem ripple */
.btn {
  position: relative;
  overflow: hidden;
  transition: background 0.2s ease, transform 0.1s ease;
}

.btn::after {
  content: '';
  position: absolute;
  inset: 0;
  background: rgba(255,255,255,0.1);
  opacity: 0;
  transition: opacity 0.2s ease;
}

.btn:hover::after  { opacity: 1; }
.btn:active { transform: scale(0.97); }

/* Input focus z animowanym labelem */
.form-group {
  position: relative;
}

.form-group label {
  position: absolute;
  top: 14px;
  left: 16px;
  color: var(--text-muted);
  font-size: 1rem;
  transition:
    top 0.2s var(--ease-smooth),
    font-size 0.2s var(--ease-smooth),
    color 0.2s ease;
  pointer-events: none;
}

.form-group input:focus + label,
.form-group input:not(:placeholder-shown) + label {
  top: -10px;
  font-size: 0.75rem;
  color: var(--accent-primary);
}

Animacje wejścia elementów

/* Fade in z dołu */
@keyframes slide-up {
  from {
    opacity: 0;
    transform: translateY(32px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Fade in z lewej */
@keyframes slide-right {
  from {
    opacity: 0;
    transform: translateX(-32px);
  }
  to {
    opacity: 1;
    transform: translateX(0);
  }
}

/* Skalowanie */
@keyframes scale-in {
  from {
    opacity: 0;
    transform: scale(0.9);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}

/* Użycie z IntersectionObserver */
.animate-on-scroll {
  opacity: 0;
}

.animate-on-scroll.visible {
  animation: slide-up 0.6s var(--ease-snappy) forwards;
}

/* Kaskadowe opóźnienia */
.stagger-item {
  opacity: 0;
}

.stagger-item.visible {
  animation: slide-up 0.5s var(--ease-snappy) both;
}

.stagger-item:nth-child(1) { animation-delay: 0.0s; }
.stagger-item:nth-child(2) { animation-delay: 0.1s; }

Animacja licznika

/* CSS-only licznik z @property */
@property --num {
  syntax: '';
  initial-value: 0;
  inherits: false;
}

.counter {
  --num: 0;
  counter-reset: num var(--num);
  animation: count-up 2s ease-out forwards;
}

.counter::after {
  content: counter(num);
}

@keyframes count-up {
  to { --num: 1250; }
}

View Transitions API

/* Natywne przejścia między stronami / stanami */

/* CSS */
::view-transition-old(root) {
  animation: 0.3s ease both slide-out;
}

::view-transition-new(root) {
  animation: 0.3s ease both slide-in;
}

@keyframes slide-out {
  to { transform: translateX(-100%); opacity: 0; }
}

@keyframes slide-in {
  from { transform: translateX(100%); opacity: 0; }
}

/* JavaScript — wyzwolenie przejścia */
document.startViewTransition(() => {
  updateDOM(); /* zmiana treści */
});

Scroll-driven animations

/* Animacja napędzana scrollem — bez JavaScript! */
@keyframes progress-bar {
  from { transform: scaleX(0); }
  to   { transform: scaleX(1); }
}

.reading-progress {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 3px;
  background: var(--accent-primary);
  transform-origin: left;
  animation: progress-bar linear;
  animation-timeline: scroll(root block);
}

/* Fade in przy scrollowaniu */
@keyframes fade-in-scroll {
  from { opacity: 0; transform: translateY(40px); }
  to   { opacity: 1; transform: translateY(0); }
}

.section {
  animation: fade-in-scroll linear both;
  animation-timeline: view();
  animation-range: entry 0% entry 30%;
}
💡 Złota zasada animacji: Czas trwania 200–300ms dla mikro-interakcji (hover, focus), 400–600ms dla wejść elementów, maksymalnie 1000ms dla złożonych przejść stron. Zbyt wolne animacje frustrują użytkownika bardziej niż ich brak.
