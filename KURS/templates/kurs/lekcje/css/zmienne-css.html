Zmienne CSS (Custom Properties)

Zmienne CSS — zwane też Custom Properties — pozwalają przechowywać wartości i reużywać je w całym arkuszu stylów. To fundament każdego nowoczesnego design systemu.

Deklaracja i użycie

/* Deklaracja — zawsze zaczyna się od -- */
:root {
  --color-primary: #7c3aed;
  --color-secondary: #a855f7;
  --font-size-base: 1rem;
  --spacing-md: 16px;
  --border-radius: 8px;
}

/* Użycie — funkcja var() */
button {
  background: var(--color-primary);
  padding: var(--spacing-md);
  border-radius: var(--border-radius);
  font-size: var(--font-size-base);
}

Wartość zapasowa (fallback)

/* Drugi argument var() to wartość zapasowa */
color: var(--color-accent, #7c3aed);
font-size: var(--font-size-xl, 1.5rem);

/* Zagnieżdżone fallbacki */
color: var(--color-brand, var(--color-primary, #7c3aed));

Zasięg zmiennych

/* :root — globalne (dostępne wszędzie) */
:root {
  --gap: 24px;
}

/* Lokalne — tylko w danym komponencie i jego dzieciach */
.card {
  --card-padding: 20px;
  --card-bg: rgba(22, 22, 31, 0.8);
  padding: var(--card-padding);
  background: var(--card-bg);
}

/* Nadpisanie lokalne */
.card.card--compact {
  --card-padding: 12px; /* nadpisuje tylko w tym kontekście */
}

Dynamiczna zmiana przez JavaScript

// Odczyt zmiennej CSS
const root = document.documentElement;
const color = getComputedStyle(root)
  .getPropertyValue('--color-primary').trim();

// Ustawienie zmiennej CSS
root.style.setProperty('--color-primary', '#e11d48');

// Na konkretnym elemencie
const card = document.querySelector('.card');
card.style.setProperty('--card-padding', '8px');

Dark Mode z zmiennymi CSS

:root {
  --bg: #ffffff;
  --text: #111111;
  --border: #e5e7eb;
}

[data-theme="dark"] {
  --bg: #0a0a0f;
  --text: #f1f5f9;
  --border: #1e1e2e;
}

body {
  background: var(--bg);
  color: var(--text);
  border-color: var(--border);
}
// Przełącznik motywu
const toggle = document.querySelector('#theme-toggle');
toggle.addEventListener('click', () => {
  const current = document.documentElement.dataset.theme;
  document.documentElement.dataset.theme =
    current === 'dark' ? 'light' : 'dark';
});

Zmienne w obliczeniach calc()

:root {
  --spacing-unit: 8px;
  --columns: 3;
  --gap: 24px;
}

.item {
  width: calc((100% - var(--gap) * (var(--columns) - 1)) / var(--columns));
  padding: calc(var(--spacing-unit) * 2);
  margin-bottom: calc(var(--spacing-unit) * 3);
}

Animacja zmiennych CSS

/* Zmienne CSS nie animują się bezpośrednio, ale można animować właściwości */
.btn {
  --glow-opacity: 0;
  box-shadow: 0 0 20px rgba(124, 58, 237, var(--glow-opacity));
  transition: box-shadow 0.3s ease;
}

.btn:hover {
  --glow-opacity: 0.6;
}

Przykład kompletnego design systemu

:root {
  /* Kolory */
  --clr-bg-primary:   #0a0a0f;
  --clr-bg-card:      #16161f;
  --clr-accent:       #7c3aed;
  --clr-accent-light: #a855f7;
  --clr-text:         #f1f5f9;
  --clr-text-muted:   #94a3b8;

  /* Typografia */
  --font-sans: 'Inter', sans-serif;
  --font-mono: 'JetBrains Mono', monospace;
  --text-sm:   0.875rem;
  --text-base: 1rem;
  --text-lg:   1.125rem;
  --text-xl:   1.25rem;
  --text-2xl:  1.5rem;

  /* Spacing */
  --space-1: 4px;
  --space-2: 8px;
  --space-3: 12px;
  --space-4: 16px;
  --space-6: 24px;
  --space-8: 32px;

  /* Radii */
  --radius-sm: 4px;
  --radius-md: 8px;
  --radius-lg: 16px;
  --radius-full: 9999px;

  /* Shadows */
  --shadow-sm: 0 1px 3px rgba(0,0,0,0.3);
  --shadow-md: 0 4px 16px rgba(0,0,0,0.4);
  --shadow-glow: 0 0 20px rgba(124,58,237,0.4);
}
💡 Konwencja nazewnictwa: Używaj nazw semantycznych zamiast opisowych — --color-primary zamiast --color-purple. Gdy zmienisz kolor marki z fioletowego na niebieski, nie musisz zmieniać nazwy zmiennej w całym projekcie.
