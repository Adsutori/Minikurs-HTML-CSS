Zaawansowany CSS Grid

Po opanowaniu podstaw Grid czas na techniki, które rozwiązują prawdziwe problemy layoutu — gęste upakowanie elementów, subgrid, nakładające się komórki i dynamiczne siatki bez media queries.

grid-auto-flow: dense

/* Domyślnie — elementy wypełniają po kolei, zostawiając dziury */
.gallery {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 16px;
}

/* dense — wypełnia dziury mniejszymi elementami */
.gallery {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-flow: dense;
  gap: 16px;
}

.gallery .featured {
  grid-column: span 2;
  grid-row: span 2;
}

Implicit grid — automatyczne wiersze

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  /* Wiersze tworzone automatycznie */
  grid-auto-rows: 200px;         /* każdy auto-wiersz ma 200px */
  grid-auto-rows: minmax(150px, auto); /* min 150px, rośnie z treścią */
}

Nakładające się elementy

/* Elementy mogą zajmować te same komórki */
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: 1fr 1fr;
}

.background-image {
  grid-column: 1 / -1;
  grid-row: 1 / -1;
  z-index: 0;
}

.content {
  grid-column: 1 / 2;
  grid-row: 1 / 2;
  z-index: 1;
  /* Nakłada się na background-image */
}

subgrid — dziedziczenie siatki

/* Dziecko używa linii siatki rodzica */
.parent {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 24px;
}

.child {
  grid-column: span 2;
  display: grid;
  grid-template-columns: subgrid; /* dziedziczy 2 kolumny rodzica */
}

/* Wyrównanie elementów w kartach */
.cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

.card {
  display: grid;
  grid-template-rows: subgrid;
  grid-row: span 3; /* tytuł | treść | stopka */
}
/* Wszystkie karty mają wyrównane tytuły, treści i stopki! */

Responsywna galeria bez media queries

/* Magiczna linia — siatka sama decyduje ile kolumn */
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 24px;
}

/* Wariant — minimum 200px, maksimum 1fr */
.cards {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: 16px;
}

Masonry layout (eksperymentalny)

/* Natywny masonry — jeszcze nie wszędzie obsługiwany */
.masonry {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: masonry;
  gap: 16px;
}

/* Fallback z columns */
.masonry-fallback {
  columns: 3;
  column-gap: 16px;
}

.masonry-fallback .item {
  break-inside: avoid;
  margin-bottom: 16px;
}

Zaawansowany layout aplikacji

/* Holy Grail Layout */
.app {
  display: grid;
  grid-template-columns: 240px 1fr 280px;
  grid-template-rows: 64px 1fr 48px;
  grid-template-areas:
    "nav    header  header"
    "nav    main    aside"
    "nav    footer  footer";
  min-height: 100vh;
}

@media (max-width: 1024px) {
  .app {
    grid-template-columns: 240px 1fr;
    grid-template-areas:
      "nav    header"
      "nav    main"
      "nav    footer";
  }
  .aside { display: none; }
}

@media (max-width: 768px) {
  .app {
    grid-template-columns: 1fr;
    grid-template-rows: 64px 1fr 48px;
    grid-template-areas:
      "header"
      "main"
      "footer";
  }
  .nav { display: none; }
}
