Flexbox

Flexbox — Flexible Box Layout — to jednowymiarowy system układu CSS. Idealny do rozmieszczania elementów w wierszu lub kolumnie, wyrównywania ich i rozdzielania przestrzeni. Rozwiązuje problemy z układem, które wcześniej wymagały hacków z float czy inline-block.

Włączenie Flexbox

.container {
  display: flex; /* lub inline-flex */
}

Element z display: flex staje się flex container. Jego bezpośrednie dzieci stają się flex items.

Oś główna i poprzeczna

Flexbox operuje na dwóch osiach: głównej (main axis) i poprzecznej (cross axis). Domyślnie oś główna biegnie poziomo (→).

flex-direction — kierunek osi

.container {
  flex-direction: row;            /* → domyślne */
  flex-direction: row-reverse;    /* ← */
  flex-direction: column;         /* ↓ */
  flex-direction: column-reverse; /* ↑ */
}

justify-content — wyrównanie na osi głównej

.container {
  justify-content: flex-start;    /* domyślne — do początku */
  justify-content: flex-end;      /* do końca */
  justify-content: center;        /* wyśrodkowane */
  justify-content: space-between; /* równe odstępy, bez marginesów */
  justify-content: space-around;  /* równe odstępy z marginesami */
  justify-content: space-evenly;  /* wszystkie odstępy równe */
}

align-items — wyrównanie na osi poprzecznej

.container {
  align-items: stretch;     /* domyślne — rozciąga do wysokości kontenera */
  align-items: flex-start;  /* do góry */
  align-items: flex-end;    /* do dołu */
  align-items: center;      /* wyśrodkowane pionowo */
  align-items: baseline;    /* wg linii bazowej tekstu */
}

Idealne centrowanie

.center-everything {
  display: flex;
  justify-content: center; /* poziomo */
  align-items: center;     /* pionowo */
  min-height: 100vh;
}

flex-wrap — zawijanie

.container {
  flex-wrap: nowrap;       /* domyślne — wszystko w jednej linii */
  flex-wrap: wrap;         /* zawija do nowych linii */
  flex-wrap: wrap-reverse; /* zawija odwrotnie */
}

gap — odstępy

.container {
  gap: 16px;          /* równy odstęp we wszystkich kierunkach */
  gap: 16px 24px;     /* wiersz | kolumna */
  row-gap: 16px;
  column-gap: 24px;
}

Właściwości flex items

flex-grow — rozrastanie

.item { flex-grow: 0; }  /* domyślne — nie rośnie */
.item { flex-grow: 1; }  /* zajmuje dostępną przestrzeń */

/* Proporcje */
.sidebar { flex-grow: 1; }
.main    { flex-grow: 3; } /* 3x więcej przestrzeni niż sidebar */

flex-shrink — kurczenie

.item { flex-shrink: 1; }  /* domyślne — kurczy się gdy brak miejsca */
.item { flex-shrink: 0; }  /* nie kurczy się nigdy */

flex-basis — bazowy rozmiar

.item { flex-basis: auto; }   /* domyślne — rozmiar treści */
.item { flex-basis: 200px; }  /* bazowa szerokość 200px */
.item { flex-basis: 30%; }

Skrócony zapis flex

/* flex: grow shrink basis */
.item { flex: 0 1 auto; }  /* domyślne */
.item { flex: 1; }         /* = flex: 1 1 0 — rośnie i kurczy się */
.item { flex: auto; }      /* = flex: 1 1 auto */
.item { flex: none; }      /* = flex: 0 0 auto — sztywny */

align-self — indywidualne wyrównanie

.item { align-self: auto; }        /* dziedziczy align-items */
.item { align-self: flex-start; }
.item { align-self: center; }
.item { align-self: flex-end; }
.item { align-self: stretch; }

order — kolejność

.item-1 { order: 2; }  /* domyślne: 0 */
.item-2 { order: 1; }  /* wyświetlony przed item-1 */
.item-3 { order: -1; } /* wyświetlony jako pierwszy */

Praktyczny przykład — karta

.card {
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.card-footer {
  margin-top: auto; /* "pcha" stopkę na dół karty */
  display: flex;
  justify-content: space-between;
  align-items: center;
}
💡 Flexbox vs Grid: Flexbox jest jednowymiarowy — świetny do nawigacji, przycisków, kart w rzędzie. Grid jest dwuwymiarowy — świetny do całych layoutów stron. W praktyce używasz obu razem.
