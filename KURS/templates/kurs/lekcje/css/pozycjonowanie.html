Pozycjonowanie

Właściwość position kontroluje jak element jest umieszczany na stronie. Razem z właściwościami top, right, bottom, left i z-index daje pełną kontrolę nad układem.

position: static (domyślne)

div { position: static; }

Element jest w normalnym przepływie dokumentu. Właściwości top/right/bottom/left/z-index nie mają efektu.

position: relative

.box {
  position: relative;
  top: 20px;   /* przesuń 20px od swojej normalnej pozycji */
  left: 10px;
}
  • Element pozostaje w normalnym przepływie (zajmuje swoje miejsce)
  • Przesuwa się względem swojej oryginalnej pozycji
  • Tworzy kontekst pozycjonowania dla dzieci absolute

position: absolute

.parent {
  position: relative; /* kontekst dla dziecka */
}

.child {
  position: absolute;
  top: 0;
  right: 0;
  /* Pozycjonowany względem najbliższego rodzica z position != static */
}
  • Wychodzi z normalnego przepływu — nie zajmuje miejsca
  • Pozycjonowany względem najbliższego przodka z position innym niż static
  • Jeśli brak takiego przodka — pozycjonowany względem <html>

position: fixed

.navbar {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  z-index: 1000;
}

.cookie-banner {
  position: fixed;
  bottom: 0;
  left: 0;
  right: 0;
}
  • Pozycjonowany względem okna przeglądarki (viewport)
  • Pozostaje w miejscu podczas przewijania strony
  • Wychodzi z normalnego przepływu

position: sticky

.section-header {
  position: sticky;
  top: 60px; /* "przykleja się" gdy dotrze do 60px od góry */
  background: white;
  z-index: 10;
}

thead {
  position: sticky;
  top: 0; /* nagłówek tabeli przyklejony podczas przewijania */
}
  • Hybryd relative i fixed
  • Zachowuje się jak relative dopóki nie osiągnie progu
  • Potem "przykleja się" jak fixed
  • Działa tylko w obrębie swojego rodzica

z-index — warstwy

.modal-overlay {
  position: fixed;
  z-index: 1000;
}

.modal {
  position: fixed;
  z-index: 1001; /* nad overlay */
}

.tooltip {
  position: absolute;
  z-index: 9999;
}

z-index działa tylko na elementach z position innym niż static. Wyższy z-index = element na wierzchu.

Centrowanie z position: absolute

/* Klasyczna technika centrowania */
.centered {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

Typowe zastosowania

WartośćTypowe zastosowanie
relativeKontekst dla dzieci absolute, drobne korekty pozycji
absoluteTooltips, dropdowny, odznaki, nakładki
fixedNavbar, cookie banner, przycisk "wróć na górę"
stickyPrzyklejone nagłówki sekcji, sticky sidebar
💡 Najczęstszy błąd: position: absolute nie działa jak oczekujesz, bo brakuje position: relative na rodzicu. Zawsze sprawdź czy rodzic ma ustawiony kontekst pozycjonowania.
