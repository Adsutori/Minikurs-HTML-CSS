Dostępność w CSS

Dostępność (accessibility, a11y) to projektowanie stron tak, by działały dla wszystkich użytkowników — w tym osób z niepełnosprawnościami, używających czytników ekranu, klawiatury lub mających preferencje dotyczące ruchu i kontrastu.

Focus — kluczowy dla klawiatury

/* NIGDY nie rób tego */
*:focus { outline: none; } /* ❌ niszczy dostępność */

/* Zamiast tego — własny, ładny focus */
:focus-visible {
  outline: 2px solid var(--accent-primary);
  outline-offset: 3px;
  border-radius: 4px;
}

/* Dla przycisków */
.btn:focus-visible {
  outline: 2px solid var(--accent-secondary);
  outline-offset: 4px;
  box-shadow: 0 0 0 4px rgba(124, 58, 237, 0.2);
}

/* :focus-visible — tylko dla nawigacji klawiaturą, nie myszy */
/* :focus — dla wszystkich (klawiatura + mysz) */

Kontrast kolorów

/* WCAG 2.1 wymaga minimalnego kontrastu */
/* AA — tekst normalny: 4.5:1 */
/* AA — tekst duży (18px+): 3:1 */
/* AAA — tekst normalny: 7:1 */

/* Złe — zbyt niski kontrast */
.bad {
  color: #999;           /* ❌ */
  background: #fff;      /* kontrast ~2.8:1 */
}

/* Dobre */
.good {
  color: #595959;        /* ✅ */
  background: #fff;      /* kontrast ~7:1 */
}

/* Sprawdź kontrast: webaim.org/resources/contrastchecker */

prefers-reduced-motion

/* Respektuj preferencje systemowe */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}

/* Lepsze podejście — definiuj animacje warunkowo */
.animated {
  /* domyślnie — bez animacji */
}

@media (prefers-reduced-motion: no-preference) {
  .animated {
    animation: slide-up 0.6s ease forwards;
    transition: transform 0.3s ease;
  }
}

prefers-color-scheme — dark mode

:root {
  --bg: #ffffff;
  --text: #111111;
  --card: #f8fafc;
  --border: #e2e8f0;
}

@media (prefers-color-scheme: dark) {
  :root {
    --bg: #0a0a0f;
    --text: #f1f5f9;
    --card: #16161f;
    --border: #1e1e2e;
  }
}

body {
  background: var(--bg);
  color: var(--text);
}

prefers-contrast

/* Użytkownicy mogą preferować wyższy kontrast */
@media (prefers-contrast: more) {
  :root {
    --text-secondary: #f1f5f9;  /* jaśniejszy */
    --border: #6b7280;          /* wyraźniejszy */
  }

  .btn {
    border: 2px solid currentColor;
  }
}

Ukrywanie elementów dostępnie

/* Wizualnie ukryty, ale dostępny dla czytników ekranu */
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border: 0;
}

/* Ukryty dla czytników, widoczny wizualnie */
.aria-hidden {
  /* dodaj aria-hidden="true" w HTML */
}

/* Skip link — pomiń nawigację */
.skip-link {
  position: absolute;
  top: -100%;
  left: 16px;
  background: var(--accent-primary);
  color: white;
  padding: 8px 16px;
  border-radius: 0 0 8px 8px;
  z-index: 9999;
  transition: top 0.2s;
}

.skip-link:focus {
  top: 0;
}

Rozmiary dotyku

/* Minimalne rozmiary dla elementów interaktywnych */
/* WCAG 2.5.5 — minimum 44x44px */
.btn,
a,
input[type="checkbox"],
input[type="radio"] {
  min-width: 44px;
  min-height: 44px;
}

/* Dla małych elementów — rozszerz obszar dotyku */
.icon-btn {
  position: relative;
  width: 24px;
  height: 24px;
}

.icon-btn::after {
  content: '';
  position: absolute;
  inset: -10px; /* rozszerza obszar klikalny o 10px */
}

Typografia dostępna

/* Nie blokuj powiększania tekstu */
html {
  font-size: 100%; /* NIE: font-size: 14px na html */
}

/* Używaj rem, nie px */
body { font-size: 1rem; }     /* ✅ */
body { font-size: 16px; }     /* ❌ nie skaluje z ustawieniami przeglądarki */

/* Długość linii — optymalna czytelność */
p, li {
  max-width: 65ch; /* ~65 znaków na linię */
}

/* Odstęp między liniami */
p {
  line-height: 1.5; /* minimum 1.5 dla tekstu */
}
