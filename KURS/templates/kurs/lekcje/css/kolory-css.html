Kolory w CSS

CSS oferuje wiele formatów zapisu kolorów — od prostych nazw słownych, przez szesnastkowe kody HEX, po nowoczesne funkcje oklch(). Każdy format ma swoje zastosowanie.

Nazwy kolorów

p { color: red; }
h1 { color: navy; }
div { background-color: lightgray; }
button { background: transparent; }

CSS obsługuje 140 nazwanych kolorów — od black do rebeccapurple.

HEX — szesnastkowy

/* Pełny zapis — #RRGGBB */
color: #ff0000;   /* czerwony */
color: #1a1a2e;   /* ciemny granat */
color: #ffffff;   /* biały */

/* Skrócony zapis — #RGB (gdy pary są identyczne) */
color: #f00;      /* czerwony = #ff0000 */
color: #fff;      /* biały = #ffffff */

/* Z przezroczystością — #RRGGBBAA */
color: #ff000080; /* czerwony 50% opacity */

RGB i RGBA

/* RGB — czerwony, zielony, niebieski (0–255) */
color: rgb(255, 0, 0);       /* czerwony */
color: rgb(26, 26, 46);      /* ciemny granat */

/* RGBA — z kanałem alpha (0 = przezroczysty, 1 = pełny) */
color: rgba(255, 0, 0, 0.5);    /* czerwony 50% */
color: rgba(124, 58, 237, 0.15); /* fioletowy 15% */

/* Nowoczesna składnia (CSS4) — spacje zamiast przecinków */
color: rgb(255 0 0);
color: rgb(255 0 0 / 50%);

HSL i HSLA

HSL — Hue (odcień 0–360°), Saturation (nasycenie 0–100%), Lightness (jasność 0–100%). Intuicyjny do tworzenia palet kolorów. 

/* HSL */
color: hsl(0, 100%, 50%);    /* czerwony */
color: hsl(270, 70%, 55%);   /* fioletowy */
color: hsl(120, 60%, 40%);   /* zielony */

/* HSLA — z przezroczystością */
color: hsla(270, 70%, 55%, 0.8);

/* Nowoczesna składnia */
color: hsl(270 70% 55% / 80%);

Nowoczesne formaty — oklch()

/* oklch — perceptualnie jednolita przestrzeń kolorów */
color: oklch(0.7 0.15 270);   /* fioletowy */
color: oklch(0.5 0.2 30);     /* pomarańczowy */

oklch() to przyszłość CSS — kolory wyglądają spójnie niezależnie od odcienia. Obsługiwany przez wszystkie nowoczesne przeglądarki.

Właściwości kolorów

div {
  color: #333;                    /* kolor tekstu */
  background-color: #f5f5f5;      /* kolor tła */
  border-color: #ddd;             /* kolor obramowania */
  outline-color: blue;            /* kolor outline */
  text-decoration-color: red;     /* kolor podkreślenia */
  caret-color: purple;            /* kursor w input */
}

Słowo kluczowe currentColor

/* currentColor dziedziczy wartość właściwości color */
.icon {
  color: #7c3aed;
  border: 2px solid currentColor; /* = #7c3aed */
  fill: currentColor;             /* dla SVG */
}

Przezroczystość — opacity vs alpha

/* opacity — wpływa na cały element i jego dzieci */
.modal-overlay {
  background: black;
  opacity: 0.5; /* cały element staje się półprzezroczysty */
}

/* alpha w kolorze — tylko tło, dzieci niezmienione */
.modal-overlay {
  background: rgba(0, 0, 0, 0.5); /* tylko tło */
}
💡 Praktyczna wskazówka: Do budowania design systemu używaj zmiennych CSS z kolorami w formacie HSL — łatwo tworzyć warianty (jaśniejszy/ciemniejszy) zmieniając tylko wartość jasności: hsl(270 70% 55%) → hsl(270 70% 40%).
