Transformacje CSS

Właściwość transform pozwala przesuwać, obracać, skalować i pochylać elementy bez wpływu na układ strony. Transformacje są akcelerowane sprzętowo — działają płynnie nawet na słabszych urządzeniach.

translate — przesunięcie

/* Przesuń o 50px w prawo i 20px w dół */
transform: translate(50px, 20px);

/* Tylko poziomo */
transform: translateX(50px);

/* Tylko pionowo */
transform: translateY(-20px);

/* Procentowo — względem własnego rozmiaru */
transform: translate(-50%, -50%); /* klasyczne centrowanie */

/* CSS4 — skrót */
translate: 50px 20px;

rotate — obrót

transform: rotate(45deg);    /* zgodnie z ruchem wskazówek */
transform: rotate(-90deg);   /* przeciwnie */
transform: rotate(0.5turn);  /* pół obrotu = 180deg */

/* Obrót 3D */
transform: rotateX(45deg);
transform: rotateY(180deg);
transform: rotateZ(45deg); /* = rotate(45deg) */

/* CSS4 */
rotate: 45deg;

scale — skalowanie

transform: scale(1.5);       /* 150% w obu kierunkach */
transform: scale(1.2, 0.8);  /* 120% X, 80% Y */
transform: scaleX(2);        /* tylko poziomo */
transform: scaleY(0.5);      /* tylko pionowo */
transform: scale(-1);        /* odbicie lustrzane */

/* CSS4 */
scale: 1.5;

skew — pochylenie

transform: skew(15deg);          /* pochylenie poziome */
transform: skew(15deg, 10deg);   /* X i Y */
transform: skewX(15deg);
transform: skewY(10deg);

Łączenie transformacji

/* Kolejność ma znaczenie! */
transform: translateX(100px) rotate(45deg) scale(1.2);

/* Różna kolejność = różny efekt */
transform: rotate(45deg) translateX(100px); /* inne niż powyżej */

transform-origin — punkt transformacji

/* Domyślnie: center center (50% 50%) */
transform-origin: top left;
transform-origin: 0 0;
transform-origin: 100% 100%;
transform-origin: 50% 50%;

/* Przykład — rozwijane menu */
.dropdown {
  transform-origin: top center;
  transform: scaleY(0);
  transition: transform 0.2s ease;
}
.dropdown.open {
  transform: scaleY(1);
}

Transformacje 3D

/* Perspektywa — wymagana dla efektów 3D */
.scene {
  perspective: 1000px;
}

/* Obrót karty */
.card {
  transform-style: preserve-3d;
  transition: transform 0.6s;
}

.card:hover {
  transform: rotateY(180deg);
}

/* Tył karty */
.card-back {
  backface-visibility: hidden;
  transform: rotateY(180deg);
}

matrix — zaawansowane

/* Macierz 2D — łączy wszystkie transformacje */
/* matrix(scaleX, skewY, skewX, scaleY, translateX, translateY) */
transform: matrix(1, 0, 0, 1, 50, 100);

/* Rzadko pisane ręcznie — generowane przez JS lub narzędzia */

Praktyczne przykłady

/* Efekt hover na karcie */
.card {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}
.card:hover {
  transform: translateY(-8px);
  box-shadow: 0 20px 40px rgba(0,0,0,0.3);
}

/* Ikona hamburgera → X */
.bar-middle {
  transition: opacity 0.2s, transform 0.3s;
}
.bar-top, .bar-bottom {
  transition: transform 0.3s;
}
.open .bar-middle { opacity: 0; }
.open .bar-top    { transform: rotate(45deg) translate(5px, 5px); }
.open .bar-bottom { transform: rotate(-45deg) translate(5px, -5px); }

/* Spinner */
.spinner {
  animation: spin 1s linear infinite;
}
@keyframes spin {
  to { transform: rotate(360deg); }
}
💡 Wydajność: Do animacji używaj wyłącznie transform i opacity — są akcelerowane przez GPU i nie powodują przeliczania layoutu (reflow). Unikaj animowania width, height, top, left, margin.
