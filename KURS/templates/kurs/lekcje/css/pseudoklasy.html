Pseudoklasy i pseudoelementy

Pseudoklasy pozwalają stylować elementy w zależności od ich stanu lub pozycji w dokumencie. Pseudoelementy tworzą wirtualne elementy, których nie ma w HTML. Razem dają ogromne możliwości bez dodatkowego kodu.

Pseudoklasy stanowe

/* Najechanie myszką */
a:hover { color: white; }
button:hover { background: var(--accent-secondary); }

/* Kliknięcie / aktywny */
button:active { transform: scale(0.97); }

/* Focus — klawiatura lub kliknięcie */
input:focus {
  outline: 2px solid var(--accent-primary);
  outline-offset: 2px;
}

/* Focus tylko z klawiatury (nie myszy) */
button:focus-visible {
  outline: 2px solid var(--accent-primary);
}

/* Odwiedzony link */
a:visited { color: #a78bfa; }

/* Niezaznaczony / zaznaczony */
input:checked + label { color: var(--accent-primary); }

/* Wyłączony */
input:disabled { opacity: 0.5; cursor: not-allowed; }
button:disabled { pointer-events: none; }

/* Wymagany */
input:required { border-color: orange; }

/* Poprawny / niepoprawny */
input:valid   { border-color: green; }
input:invalid { border-color: red; }

Pseudoklasy strukturalne

/* Pierwsze i ostatnie dziecko */
li:first-child { border-top: none; }
li:last-child  { border-bottom: none; }

/* Nth-child — co n-ty element */
tr:nth-child(even) { background: rgba(255,255,255,0.03); } /* parzyste */
tr:nth-child(odd)  { background: transparent; }            /* nieparzyste */
li:nth-child(3)    { color: gold; }                        /* dokładnie 3. */
li:nth-child(3n)   { color: gold; }                        /* co 3. */
li:nth-child(3n+1) { color: red; }                         /* 1, 4, 7, ... */

/* Ostatnie nth */
li:nth-last-child(2) { /* drugi od końca */ }

/* Jedyne dziecko */
p:only-child { font-size: 1.2rem; }

/* Pierwsze/ostatnie tego typu */
p:first-of-type { font-weight: 600; }
p:last-of-type  { margin-bottom: 0; }

/* Nth tego typu */
p:nth-of-type(2) { color: var(--accent-secondary); }

/* Pusty element */
div:empty { display: none; }

/* Nie pasuje do selektora */
li:not(.active) { opacity: 0.7; }
li:not(:last-child) { border-bottom: 1px solid var(--border); }

Pseudoklasy relacyjne

/* :is() — grupowanie selektorów */
:is(h1, h2, h3, h4) {
  font-family: 'Inter', sans-serif;
  line-height: 1.2;
}

/* :where() — jak :is() ale zerowa specyficzność */
:where(h1, h2, h3) { margin-bottom: 1rem; }

/* :has() — rodzic zawierający dany element */
.card:has(img) { padding: 0; }
form:has(input:invalid) .submit-btn { opacity: 0.5; }
li:has(> a.active) { background: rgba(124,58,237,0.1); }

Pseudoelementy

/* ::before i ::after — wirtualne elementy przed/po treści */
.btn::before {
  content: '→ ';
  color: var(--accent-secondary);
}

.required::after {
  content: ' *';
  color: red;
}

/* Dekoracyjna linia pod nagłówkiem */
h2::after {
  content: '';
  display: block;
  width: 60px;
  height: 3px;
  background: var(--accent-primary);
  margin-top: 8px;
  border-radius: 2px;
}

/* ::first-line — pierwsza linia tekstu */
p::first-line {
  font-weight: 600;
  color: var(--text-primary);
}

/* ::first-letter — pierwsza litera (drop cap) */
p::first-letter {
  font-size: 3rem;
  font-weight: 700;
  float: left;
  line-height: 1;
  margin-right: 8px;
  color: var(--accent-primary);
}

/* ::selection — zaznaczony tekst */
::selection {
  background: rgba(124, 58, 237, 0.4);
  color: white;
}

/* ::placeholder */
input::placeholder {
  color: var(--text-muted);
  font-style: italic;
}

/* ::marker — marker listy */
li::marker {
  color: var(--accent-primary);
  font-size: 1.2em;
}

Praktyczny przykład — tooltip z CSS

.tooltip {
  position: relative;
  cursor: help;
}

.tooltip::after {
  content: attr(data-tip); /* pobiera wartość atrybutu data-tip */
  position: absolute;
  bottom: calc(100% + 8px);
  left: 50%;
  transform: translateX(-50%);
  background: #1e1e2e;
  color: white;
  padding: 6px 12px;
  border-radius: 6px;
  font-size: 0.8rem;
  white-space: nowrap;
  opacity: 0;
  pointer-events: none;
  transition: opacity 0.2s;
}

.tooltip:hover::after {
  opacity: 1;
}
💡 Pamiętaj: ::before i ::after wymagają właściwości content — nawet pustej (content: ''). Bez niej pseudoelement nie zostanie wyrenderowany.
