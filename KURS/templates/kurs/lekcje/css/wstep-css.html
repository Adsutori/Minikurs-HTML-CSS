Wstęp do CSS

CSS — Cascading Style Sheets — to język odpowiedzialny za wygląd stron internetowych. HTML buduje strukturę, CSS nadaje jej kolory, czcionki, układy i animacje. Bez CSS każda strona wyglądałaby jak czarno-biały dokument tekstowy.

Czym jest CSS?

CSS to język arkuszy stylów. Opisuje jak elementy HTML mają być wyświetlane — ich rozmiar, kolor, pozycję, odstępy i wiele więcej. Działa na zasadzie reguł: „znajdź ten element i nadaj mu te właściwości".

Krótka historia

  • 1994 — Håkon Wium Lie proponuje CSS jako rozwiązanie dla stylowania stron
  • 1996 — CSS1 staje się oficjalnym standardem W3C
  • 1998 — CSS2 dodaje pozycjonowanie i media types
  • 2011 — CSS3 wprowadza moduły: animacje, Flexbox, gradienty
  • Dziś — CSS jest żywym standardem, nowe właściwości dodawane są na bieżąco

Jak CSS łączy się z HTML?

Są trzy sposoby dołączenia CSS do strony:

1. Zewnętrzny plik (najlepsza praktyka)

<head>
  <link rel="stylesheet" href="style.css">
</head>

2. Style wewnętrzne w <style>

<head>
  <style>
    p { color: red; }
  </style>
</head>

3. Style inline (na elemencie)

<p style="color: red">Tekst</p>
MetodaZaletyWady
Zewnętrzny plik Reużywalny, łatwy w utrzymaniu Dodatkowe żądanie HTTP
Wewnętrzny <style> Szybki do testowania Nie reużywalny między stronami
Inline Najwyższy priorytet Trudny w utrzymaniu, brak pseudoklas

Pierwsza reguła CSS

/* To jest komentarz w CSS */

p {
  color: navy;
  font-size: 18px;
  line-height: 1.6;
}

Reguła CSS składa się z selektora (p), nawiasów klamrowych i deklaracji w formacie właściwość: wartość;.

💡 CSS = Kaskada. Słowo "kaskadowe" oznacza, że style nakładają się na siebie warstwami — styl przeglądarki, potem Twój arkusz, potem style inline. Każda warstwa może nadpisać poprzednią.
