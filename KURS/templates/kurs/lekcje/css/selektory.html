Selektory CSS

Selektor to część reguły CSS, która mówi przeglądarce do którego elementu zastosować styl. Znajomość selektorów to fundament pisania efektywnego CSS.

Selektory podstawowe

Selektor elementu

/* Wszystkie akapity na stronie */
p {
  color: #333;
  line-height: 1.6;
}

/* Wszystkie nagłówki h2 */
h2 {
  font-size: 1.5rem;
  font-weight: 700;
}

Selektor klasy

/* Element z class="btn" */
.btn {
  padding: 12px 24px;
  border-radius: 8px;
  cursor: pointer;
}

/* Można łączyć — p z klasą .intro */
p.intro {
  font-size: 1.2rem;
  font-weight: 500;
}

Selektor ID

/* Element z id="header" */
#header {
  background: #1a1a2e;
  height: 60px;
}

Selektor uniwersalny

/* Wszystkie elementy na stronie */
* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

Selektory złożone

Selektor potomka (spacja)

/* Każdy <a> wewnątrz <nav> — na dowolnym poziomie */
nav a {
  text-decoration: none;
  color: white;
}

Selektor bezpośredniego dziecka (>)

/* Tylko <li> będące bezpośrednim dzieckiem <ul> */
ul > li {
  list-style: disc;
  padding: 4px 0;
}

Selektor sąsiada (+)

/* <p> bezpośrednio po <h2> */
h2 + p {
  margin-top: 8px;
  color: #666;
}

Selektor rodzeństwa (~)

/* Wszystkie <p> po <h2> (nie tylko bezpośredni) */
h2 ~ p {
  color: #555;
}

Selektory atrybutów

/* Element z atrybutem href */
a[href] { color: blue; }

/* Dokładna wartość atrybutu */
input[type="email"] { border-color: green; }

/* Atrybut zaczyna się od "https" */
a[href^="https"] { color: green; }

/* Atrybut kończy się na ".pdf" */
a[href$=".pdf"] { color: red; }

/* Atrybut zawiera "google" */
a[href*="google"] { font-weight: bold; }

Grupowanie selektorów

/* Ten sam styl dla wielu selektorów — oddziel przecinkiem */
h1,
h2,
h3,
h4 {
  font-family: 'Inter', sans-serif;
  line-height: 1.2;
  color: #111;
}

Specyficzność selektorów

Gdy kilka reguł pasuje do tego samego elementu, wygrywa ta o wyższej specyficzności:

SelektorPunktyPrzykład
Inline style1000style="color:red"
ID100#header
Klasa / atrybut / pseudoklasa10.btn, [type], :hover
Element / pseudoelement1p, ::before
Uniwersalny *0*
💡 Zasada: Unikaj używania ID w CSS — mają zbyt wysoką specyficzność i trudno je nadpisać. Używaj klas. Nigdy nie używaj !important jako rozwiązania problemów ze specyficznością — to tylko maskuje prawdziwy problem.
