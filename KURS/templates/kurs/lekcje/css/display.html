Właściwość display

display to jedna z najważniejszych właściwości CSS — decyduje o tym, jak element jest renderowany i jak wchodzi w interakcję z innymi elementami. Zmiana display całkowicie zmienia zachowanie elementu.

display: block

div { display: block; }
  • Zajmuje całą dostępną szerokość
  • Zaczyna się od nowej linii
  • Respektuje width, height, marginesy pionowe
  • Domyślne dla: div, p, h1–h6, section, article

display: inline

span { display: inline; }
  • Zajmuje tylko tyle miejsca ile potrzebuje treść
  • Nie zaczyna nowej linii — płynie z tekstem
  • Ignoruje width, height, marginesy pionowe
  • Domyślne dla: span, a, strong, em, img

display: inline-block

button { display: inline-block; }
  • Płynie z tekstem jak inline
  • Ale respektuje width, height i marginesy pionowe jak block
  • Idealne dla przycisków, odznak, elementów nawigacji

display: none

.hidden { display: none; }
  • Całkowicie usuwa element z układu strony
  • Element nie zajmuje miejsca — jakby nie istniał
  • Różni się od visibility: hidden — tamto ukrywa ale zachowuje miejsce

display: flex

.container {
  display: flex;
  gap: 16px;
  align-items: center;
  justify-content: space-between;
}

Włącza Flexbox — jednowymiarowy system układu (oś pozioma lub pionowa). Omówiony szczegółowo w lekcji Flexbox.

display: grid

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
}

Włącza CSS Grid — dwuwymiarowy system układu (wiersze i kolumny). Omówiony szczegółowo w lekcji CSS Grid.

display: inline-flex i inline-grid

/* Flex/Grid który płynie z tekstem jak inline */
.badge { display: inline-flex; align-items: center; gap: 4px; }
.mini-grid { display: inline-grid; grid-template-columns: 1fr 1fr; }

display: contents

/* Element "znika" — jego dzieci zachowują się jakby były bezpośrednio w rodzicu */
.wrapper { display: contents; }

visibility vs display: none

/* Ukrywa element ale zachowuje jego miejsce w układzie */
.invisible { visibility: hidden; }

/* Całkowicie usuwa z układu — nie zajmuje miejsca */
.gone { display: none; }

/* Przezroczysty ale widoczny dla eventów */
.transparent { opacity: 0; }
