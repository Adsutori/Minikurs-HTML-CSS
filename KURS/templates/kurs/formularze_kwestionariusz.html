O projekcie

Projekt składa się z trzech plików HTML: strony głównej (index.html), formularza logowania (login.html) i rejestracji (register.html) oraz panelu z kwestionariuszem (dashboard.html). Dane przechowywane są w localStorage przeglądarki — bez backendu.

Formularz kwestionariusza podzielony jest na 3 grupy i 7 podgrup z użyciem <fieldset> i <legend>. Każda podgrupa stosuje inny styl etykiet — klasyczne <label>, pływające etykiety CSS lub etykiety inline przy radio/checkbox.

📋 Formularz — struktura

Minimum 3 grupy — każda to <fieldset class="group"> z <legend>; projekt ma 3 grupy: Dane Osobowe/Profil, Wykształcenie i Praca, Dodatkowe Informacje
Po 2 podgrupy w każdej grupie — zagnieżdżone <fieldset class="subgroup">; każda podgrupa ma minimum 3 pola
Różne style etykiet w podgrupach — style-a: klasyczne <label> nad polem; style-b: pływające etykiety CSS (floating labels); style-c: etykiety z ikonami Lucide

Oznaczenie pól wymaganych — gwiazdka * przy każdym obowiązkowym polu; atrybut required w HTML
Używamy: label, fieldset, legend — każde pole ma <label for="..."> powiązany przez id

2 Toggle hasła — uwidacznianie / ukrywanie 

<!-- HTML -->
<div class="password-wrapper">
  <input type="password" id="haslo">
  <button type="button" class="toggle-pass"
          data-target="haslo">
    <i data-lucide="eye" id="toggleIcon-haslo"></i>
  </button>
</div>

// JavaScript
document.querySelectorAll('.toggle-pass').forEach(function(btn) {
  btn.addEventListener('click', function() {
    var id    = this.getAttribute('data-target');
    var input = document.getElementById(id);
    var icon  = document.getElementById('toggleIcon-' + id);
    input.type = input.type === 'password' ? 'text' : 'password';
    icon.setAttribute('data-lucide',
      input.type === 'password' ? 'eye' : 'eye-off');
    lucide.createIcons();
  });
});

3 Walidacja siły hasła — pasek postępu 

function ocenHaslo(val) {
    var s = 0;
    if (val.length >= 8)                                     s++;
    if (/[A-Z]/.test(val))                                   s++;
    if (/[0-9]/.test(val))                                   s++;
    if (/[!@#$%^&*()_+\-=\[\]{};':"\\|,.<>\/?]/.test(val)) s++;
    return s; // 0-4: im więcej tym silniejsze
  }

  document.getElementById('haslo').addEventListener('input', function() {
    var s     = ocenHaslo(this.value);
    var bar   = document.getElementById('strengthBar');
    var label = document.getElementById('strengthLabel');
    bar.className = 'password-strength-bar';
    if      (s <= 1) { bar.classList.add('strength-weak');
                       label.textContent = 'Słabe'; }
    else if (s <= 3) { bar.classList.add('strength-medium');
                       label.textContent = 'Średnie'; }
    else            { bar.classList.add('strength-strong');
                       label.textContent = 'Silne ✓'; }
  });

4 Skrót PESEL — abbr + walidacja cyfr 

<!-- abbr rozwijający skrót po najechaniu -->
<label for="pesel">
  <abbr title="Powszechny Elektroniczny System Ewidencji Ludności">
    PESEL
  </abbr>
  <span class="required">*</span>
</label>
<input type="text" id="pesel"
       pattern="[0-9]{11}"
       maxlength="11"
       inputmode="numeric"
       required>

// Walidacja JS — tylko cyfry, dokładnie 11
document.getElementById('pesel').addEventListener('input', function() {
  this.value = this.value.replace(/\D/g, '');
  document.getElementById('peselError').textContent =
    (this.value.length > 0 && this.value.length !== 11)
      ? 'PESEL musi mieć dokładnie 11 cyfr.' : '';
});

5 Pływające etykiety — floating labels CSS 

/* CSS — label unosi się gdy pole ma focus lub wartość */
.float-group {
  position: relative;
}
.float-group input {
  padding-top: 1.4rem;   /* miejsce na etykietę */
}
.float-group label {
  position:   absolute;
  top:        1rem;
  left:       0.75rem;
  transition: all 0.2s ease;
  pointer-events: none;
  color:      var(--text-muted);
}
/* Gdy input ma focus LUB nie jest pusty (placeholder=" ") */
.float-group input:focus + label,
.float-group input:not(:placeholder-shown) + label {
  top:       0.3rem;
  font-size: 0.7rem;
  color:     var(--accent-secondary);
}

<!-- HTML — label MUSI być po input, placeholder=" " (spacja!) -->
<div class="form-field float-group">
  <input type="text" id="ulica" placeholder=" ">
  <label for="ulica">Ulica</label>
</div>
