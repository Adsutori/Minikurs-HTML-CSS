Wstęp do HTML

HTML (HyperText Markup Language) to język znaczników używany do tworzenia struktury stron internetowych. Nie jest językiem programowania — to język opisu treści. Przeglądarka czyta plik HTML i na jego podstawie wyświetla tekst, obrazy, linki i inne elementy.

HTML5 to piąta i aktualna wersja standardu, wprowadzona w 2014 roku. Dodała semantyczne znaczniki (<header>, <nav>, <article>), natywne multimedia (<video>, <audio>) oraz nowe typy formularzy.

Szkielet dokumentu HTML5

<!-- Deklaracja typu dokumentu — ZAWSZE pierwsza linia -->
<!DOCTYPE html>

<!-- Korzeń dokumentu, atrybut lang określa język -->
<html lang="pl">

<head>
  <!-- Kodowanie znaków — zawsze UTF-8 -->
  <meta charset="UTF-8">

  <!-- Responsywność na urządzeniach mobilnych -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- Tytuł widoczny w zakładce przeglądarki -->
  <title>Moja pierwsza strona</title>

  <!-- Podłączenie arkusza CSS -->
  <link rel="stylesheet" href="style.css">
</head>

<body>
  <!-- Cała widoczna treść strony -->
  <h1>Witaj, świecie!</h1>
  <p>To jest moja pierwsza strona HTML.</p>

  <!-- Skrypty JS na końcu body -->
  <script src="script.js"></script>
</body>

</html>

Tekst

Znaczniki tekstowe pozwalają nadawać treści strukturę i znaczenie semantyczne.

<!-- Nagłówki — h1 najważniejszy, h6 najmniej ważny -->
<h1>Nagłówek pierwszego poziomu</h1>
<h2>Nagłówek drugiego poziomu</h2>
<h3>Nagłówek trzeciego poziomu</h3>

<!-- Akapit -->
<p>To jest zwykły akapit tekstu.</p>

<!-- Formatowanie inline -->
<p>
  <strong>Pogrubienie (ważne)</strong>,
  <em>kursywa (nacisk)</em>,
  <mark>zaznaczenie</mark>,
  <del>przekreślenie</del>,
  <ins>wstawienie</ins>,
  <code>kod inline</code>
</p>

<!-- Łamanie linii i linia pozioma -->
Pierwsza linia<br>
Druga linia
<hr>

<!-- Cytat blokowy -->
<blockquote cite="https://example.com">
  Każdy może nauczyć się kodować.
</blockquote>

<!-- Preformatowany tekst -->
<pre>
  Ten tekst zachowuje
    spacje i wcięcia
</pre>

Inputy

Element <input> jest najbardziej wszechstronnym elementem formularzy. Jego zachowanie zmienia się całkowicie w zależności od atrybutu type.

type= Opis Przykład użycia
"text"Zwykłe pole tekstoweImię, nazwisko
"password"Hasło (znaki ukryte)Logowanie
"email"Adres e-mail (walidacja)Formularz kontaktowy
"number"Liczba (min/max/step)Wiek, ilość
"date"Data (kalendarz)Data urodzenia
"time"GodzinaRezerwacja
"color"Wybór koloruUstawienia wyglądu
"range"Suwak (min/max)Głośność, ocena
"checkbox"Pole wyboru (wielokrotny)Zgody, umiejętności
"radio"Przycisk opcji (jeden z grupy)Płeć, poziom
"file"Wybór plikuUpload zdjęcia
"submit"Przycisk wysyłaniaWyślij formularz
"reset"Przycisk resetowaniaWyczyść formularz
"hidden"Ukryte pole (niewidoczne)CSRF token, ID
"tel"Numer telefonuKontakt
"url"Adres URL (walidacja)Strona WWW
 
<!-- Przykłady atrybutów inputów -->
<input type="text"
       placeholder="Wpisz imię..."
       required
       minlength="2"
       maxlength="50">

<input type="number"
       min="0"
       max="100"
       step="5"
       value="50">

<input type="email"
       pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$"
       required>

<input type="range"
       min="1"
       max="10"
       value="5"
       data-output="range-val">
<span id="range-val">5</span>

<input type="text" disabled  value="Pole wyłączone">
<input type="text" readonly value="Tylko do odczytu">

Formularze

Formularz to kontener zbierający dane od użytkownika i wysyłający je na serwer.

<form action="/wyslij" method="POST">

  <!-- Fieldset grupuje powiązane pola -->
  <fieldset>
    <legend>Dane kontaktowe</legend>

    <!-- Label powiązany z inputem przez for/id -->
    <label for="imie">Imię:</label>
    <input type="text" id="imie" name="imie" required>

    <label for="email">E-mail:</label>
    <input type="email" id="email" name="email" required>
  </fieldset>

  <!-- Select — lista rozwijana -->
  <label for="kraj">Kraj:</label>
  <select id="kraj" name="kraj">
    <optgroup label="Europa">
      <option value="pl">Polska</option>
      <option value="de">Niemcy</option>
    </optgroup>
  </select>

  <!-- Textarea — wieloliniowe pole tekstowe -->
  <label for="wiad">Wiadomość:</label>
  <textarea id="wiad" name="wiadomosc"
             rows="5" cols="40"
             placeholder="Wpisz wiadomość..."></textarea>

  <!-- Przyciski -->
  <button type="submit">Wyślij</button>
  <button type="reset">Wyczyść</button>

</form>

Atrybuty globalne

Atrybuty globalne można stosować na każdym elemencie HTML.

AtrybutOpisPrzykład
idUnikalny identyfikator elementuid="naglowek"
classKlasa CSS (może być wiele)class="btn btn-primary"
styleStyle inline CSSstyle="color:red"
titleTooltip przy najechaniu myszątitle="Więcej info"
langJęzyk zawartości elementulang="pl"
dirKierunek tekstu (ltr/rtl)dir="rtl"
hiddenUkrywa elementhidden
tabindexKolejność fokusa klawiszem Tabtabindex="1"
data-*Własne atrybuty danychdata-id="42"
aria-*Dostępność (czytniki ekranu)aria-label="Zamknij"
 
<!-- data-* — własne atrybuty danych, dostępne przez JS -->
<button
  id="btn-kurs"
  class="btn btn-primary"
  data-kurs-id="42"
  data-poziom="podstawowy"
  aria-label="Rozpocznij kurs HTML"
  title="Kliknij, aby zacząć"
>
  Rozpocznij kurs
</button>
