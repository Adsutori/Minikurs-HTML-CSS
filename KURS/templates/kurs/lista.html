📋 Lista

Listy HTML — nieuporządkowane, uporządkowane, definicji i zagnieżdżone.

Listy to jeden z fundamentów HTML. Używa się ich do tworzenia menu nawigacyjnych, wyliczeń, kroków instrukcji, słowników i wielu innych elementów interfejsu.

🔵 Lista nieuporządkowana (ul)

Używana gdy kolejność elementów nie ma znaczenia — np. lista zakupów, menu, umiejętności.

Efekt:
  • HTML5 — struktura strony
  • CSS3 — wygląd i animacje
  • JavaScript — interaktywność
  • Python — backend i skrypty
  • Git — kontrola wersji
  • SQL — bazy danych

🔢 Lista uporządkowana (ol)

Używana gdy kolejność ma znaczenie — kroki instrukcji, ranking, przepis.

type="1" (domyślny)
  1. Otwórz edytor
  2. Napisz kod
  3. Zapisz plik
  4. Otwórz w przeglądarce
type="A"
  1. Opcja A
  2. Opcja B
  3. Opcja C
  4. Opcja D
type="I"
  1. Rozdział I
  2. Rozdział II
  3. Rozdział III
  4. Rozdział IV
<!-- Cyfry arabskie (domyślne) -->
<ol type="1">
  <li>Otwórz edytor</li>
  <li>Napisz kod</li>
</ol>

<!-- Wielkie litery -->
<ol type="A">
  <li>Opcja A</li>
</ol>

<!-- Cyfry rzymskie -->
<ol type="I">
  <li>Rozdział I</li>
</ol>

<!-- Start od innej liczby -->
<ol start="5">
  <li>Piąty element</li>
</ol>
