O projekcie

Projekt to wirtualna wyprawa przez trzy kontynenty — Europę, Azję i Afrykę — zaprezentowana w formie zagnieżdżonej listy HTML z galerią fotograficzną. Każde zdjęcie posiada tytuł nad fotografią oraz ukryty opis (5–10 zdań) wyświetlany w modalnym oknie po kliknięciu.

Projekt spełnia wymagania dodatkowe: 4 punkty główne (Europa, Azja, Afryka + czwarty kontynent do wyboru), po 4 podpunkty w każdym i po 4 zdjęcia w każdym podpunkcie — łącznie 36 grafik zamiast podstawowych 27.

  <li>Europa
    <ol type="a">             ← poziom 2: literowa (a. b. c.) — kraje
      <li>Polska
        <ul>                   ← poziom 3: nieuporządkowana — regiony/miasta
          <li>Tatry
            <ul class="photo-row"> ← poziom 4: graficzna — zdjęcia w rzędzie

                Czarny Staw Polski  ← nadpis NAD zdjęciem

                <img

1 Nawigacja z dropdownem — hover CSS 

<!-- HTML -->
<div class="nav-dropdown">
  <a href="#europa" class="nav-link">Europa</a>
  <div class="dropdown-content">
    <a href="#polska">Polska</a>
    <a href="#wlochy">Włochy</a>
    <a href="#hiszpania">Hiszpania</a>
  </div>
</div>

/* CSS — kluczowe: display:none domyślnie, block przy hover */
.nav-dropdown {
  position: relative;
  display:  inline-block;
}
.dropdown-content {
  display:   none;
  position:  absolute;
  top:       100%;
  left:      0;
  background: rgba(15,15,26,0.95);
  min-width: 140px;
  z-index:   100;
}
.nav-dropdown:hover .dropdown-content {
  display: block; /* ← to jest cały sekret dropdownu */
}

// JavaScript — otwieranie modala
function openPhoto(img) {
  document.getElementById('modalImage').src =
    img.src;
  document.getElementById('modalCaption').textContent =
    img.getAttribute('data-desc');
  document.getElementById('photoModal').style.display =
    'flex';
}

4 Płynne przewijanie do sekcji — scrollTo z uwzględnieniem navbaru 

// Scroll działa na .scroll-container, nie na window!
// Trzeba odjąć wysokość navbaru od pozycji elementu

function scrollToHash(hash) {
  const container = document.querySelector('.scroll-container');
  const navbar    = document.querySelector('.navbar');
  const target    = document.getElementById(hash.slice(1));
  if (!target) return;

  const offset = navbar.getBoundingClientRect().height + 8;
  const top    = target.getBoundingClientRect().top
               - container.getBoundingClientRect().top
               + container.scrollTop
               - offset;

  container.scrollTo({ top, behavior: 'smooth' });
}

// Przechwycenie kliknięć w linki #hash
document.addEventListener('click', (e) => {
  const a = e.target.closest('a[href^="#"]');
  if (!a) return;
  e.preventDefault();
  scrollToHash(a.getAttribute('href'));
});
