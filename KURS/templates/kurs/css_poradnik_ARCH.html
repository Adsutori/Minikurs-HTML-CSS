Wstęp do CSS

CSS (Cascading Style Sheets) to język opisu wyglądu dokumentów HTML. Słowo „kaskadowy" oznacza, że style nakładają się na siebie według określonych reguł pierwszeństwa.

Trzy sposoby dodawania CSS

/* 1. Zewnętrzny plik CSS (zalecane) */
/* W <head>: <link rel="stylesheet" href="style.css"> */

/* 2. Wewnętrzny blok <style> w <head> */
<style>
  p { color: red; }
</style>

/* 3. Inline style (unikać — trudne w utrzymaniu) */
<p style="color: red;">Tekst</p>

Selektory

/* Selektor elementu */
p { color: #94a3b8; }

/* Selektor klasy */
.btn { padding: 12px 24px; }

/* Selektor ID */
#naglowek { font-size: 2rem; }

/* Selektor atrybutu */
input[type="email"] { border-color: #7c3aed; }
a[href^="https"]    { color: green; }   /* href zaczyna się od https */
a[href$=".pdf"]     { color: red; }     /* href kończy się na .pdf */

/* Kombinatory */
div p       { color: white; }    /* potomek (dowolny poziom) */
div > p     { color: white; }    /* bezpośrednie dziecko */
h2 + p      { color: white; }    /* sąsiad (następny element) */
h2 ~ p      { color: white; }    /* wszyscy następni sąsiedzi */

/* Pseudo-klasy */
a:hover          { color: #a855f7; }
button:focus     { outline: 2px solid #7c3aed; }
li:first-child   { font-weight: bold; }
li:last-child    { border-bottom: none; }
li:nth-child(2n) { background: rgba(255,255,255,0.05); } /* parzyste */
input:valid      { border-color: green; }
input:invalid    { border-color: red; }
p:not(.wyjątek) { color: #94a3b8; }

/* Pseudo-elementy */
p::first-line   { font-weight: bold; }
p::first-letter { font-size: 2em; float: left; }
.btn::before    { content: "→ "; }
.btn::after     { content: ""; display: block; }

Box Model

Każdy element HTML jest prostokątem składającym się z czterech warstw.

margin
border
padding
content
.element {
  /* Wymiary treści */
  width:  300px;
  height: 200px;

  /* Wewnętrzny odstęp (od treści do border) */
  padding: 16px;              /* wszystkie strony */
  padding: 8px 16px;         /* góra/dół  lewo/prawo */
  padding: 4px 8px 12px 16px; /* góra prawo dół lewo */

  /* Obramowanie */
  border: 2px solid #7c3aed;
  border-radius: 12px;

  /* Zewnętrzny odstęp (od border do sąsiednich elementów) */
  margin: 24px auto; /* auto centruje poziomo */

  /* WAŻNE: box-sizing zmienia jak liczyć szerokość */
  box-sizing: border-box;  /* width = treść + padding + border */
  box-sizing: content-box; /* width = tylko treść (domyślne) */
}

Kolory i tła

.element {
  /* Formaty kolorów */
  color: red;                        /* nazwa */
  color: #7c3aed;                    /* hex */
  color: #7c3;                       /* hex skrócony */
  color: rgb(124, 58, 237);          /* RGB */
  color: rgba(124, 58, 237, 0.8);   /* RGB + przezroczystość */
  color: hsl(263, 83%, 58%);        /* HSL (barwa, nasycenie, jasność) */
  color: hsla(263, 83%, 58%, 0.5); /* HSL + przezroczystość */

  /* Tło */
  background-color: #0a0a0f;
  background-image: url('obraz.jpg');
  background-size:  cover;          /* cover | contain | 100% */
  background-position: center;
  background-repeat: no-repeat;
  background-attachment: fixed;    /* parallax efekt */

  /* Gradient liniowy */
  background: linear-gradient(135deg, #7c3aed, #a855f7);

  /* Gradient radialny */
  background: radial-gradient(circle, #7c3aed 0%, #0a0a0f 70%);

  /* Przezroczystość całego elementu */
  opacity: 0.8;
}

Typografia

body {
  /* Rodzina fontów — fallback chain */
  font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;

  /* Rozmiar */
  font-size: 1rem;          /* 16px domyślnie */
  font-size: clamp(1rem, 2.5vw, 1.5rem); /* responsywny rozmiar */

  /* Grubość */
  font-weight: 400;   /* 100–900, lub: thin/normal/bold */

  /* Styl */
  font-style: italic; /* normal | italic | oblique */

  /* Wysokość linii */
  line-height: 1.6;   /* bezjednostkowe — 1.6 × font-size */

  /* Odstęp między literami */
  letter-spacing: 0.05em;

  /* Odstęp między wyrazami */
  word-spacing: 4px;
}

h1 {
  /* Wyrównanie tekstu */
  text-align: center;      /* left | right | center | justify */

  /* Dekoracja */
  text-decoration: underline; /* none | underline | line-through */

  /* Transformacja */
  text-transform: uppercase; /* none | uppercase | lowercase | capitalize */

  /* Cień tekstu */
  text-shadow: 0 0 20px rgba(124, 58, 237, 0.5);

  /* Gradient tekstu */
  background: linear-gradient(135deg, #7c3aed, #a855f7);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;

Flexbox

Flexbox to jednowymiarowy system layoutu — układa elementy w wierszu lub kolumnie.

1
2
3 (flex-grow:2)
4
/* KONTENER (rodzic) */
.container {
  display: flex;

  /* Kierunek głównej osi */
  flex-direction: row;            /* row | row-reverse | column | column-reverse */

  /* Zawijanie elementów */
  flex-wrap: wrap;               /* nowrap | wrap | wrap-reverse */

  /* Wyrównanie na osi głównej (poziomej dla row) */
  justify-content: space-between; /* flex-start | flex-end | center | space-around | space-evenly */

  /* Wyrównanie na osi poprzecznej (pionowej dla row) */
  align-items: center;           /* stretch | flex-start | flex-end | center | baseline */

  /* Wyrównanie wielu wierszy */
  align-content: flex-start;

  /* Odstępy */
  gap: 16px;
  gap: 16px 24px; /* row-gap column-gap */
}

/* ELEMENTY (dzieci) */
.item {
  /* Wzrost — jak bardzo element może się rozciągnąć */
  flex-grow:   1;   /* 0 = nie rośnie, 1+ = rośnie proporcjonalnie */

  /* Kurczenie — jak bardzo element może się skurczyć */
  flex-shrink: 1;   /* 0 = nie kurczy się */

  /* Bazowy rozmiar przed podziałem przestrzeni */
  flex-basis:  200px;

  /* Skrót: grow shrink basis */
  flex: 1 1 200px;
  flex: 1;          /* = flex: 1 1 0% */

  /* Kolejność wyświetlania */
  order: 2;

  /* Nadpisanie align-items dla jednego elementu */
  align-self: flex-end;
}

CSS Grid

Grid to dwuwymiarowy system layoutu — kontroluje jednocześnie wiersze i kolumny.

header
sidebar
main
footer
/* KONTENER */
.grid {
  display: grid;

  /* Definicja kolumn */
  grid-template-columns: repeat(3, 1fr);        /* 3 równe kolumny */
  grid-template-columns: 260px 1fr 220px;       /* fixed auto fixed */
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* responsywne */

  /* Definicja wierszy */
  grid-template-rows: auto 1fr auto;  /* header content footer */

  /* Obszary nazwane */
  grid-template-areas:
    "header  header  header"
    "sidebar main    main"
    "footer  footer  footer";

  /* Odstępy */
  gap: 24px;
  row-gap: 16px;
  column-gap: 24px;
}

/* ELEMENTY */
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

/* Ręczne pozycjonowanie */
.item {
  grid-column: 1 / 3;   /* od linii 1 do linii 3 */
  grid-column: span 2;  /* zajmuje 2 kolumny */
  grid-row:    1 / 3;
}

Pozycjonowanie

/* static — domyślne, w normalnym przepływie dokumentu */
.static   { position: static; }

/* relative — przesunięty względem swojego normalnego miejsca */
.relative {
  position: relative;
  top: 10px;   /* przesuwa w dół o 10px */
  left: 20px;  /* przesuwa w prawo o 20px */
}

/* absolute — wyrwany z przepływu, pozycja względem rodzica z position != static */
.absolute {
  position: absolute;
  top: 0; right: 0;
  bottom: 0; left: 0;  /* rozciąga na cały rodzic */
}

/* fixed — względem okna przeglądarki, nie scrolluje */
.navbar {
  position: fixed;
  top: 0; left: 0; right: 0;
  z-index: 100;
}

/* sticky — jak relative, ale "przykleja się" przy scrollu */
.sidebar {
  position: sticky;
  top: 64px;  /* przykleja się 64px od góry */
}

/* z-index — kolejność warstw (wyższy = na wierzchu) */
.modal   { z-index: 1000; }
.overlay { z-index: 999; }
.navbar  { z-index: 100; }

Animacje i Przejścia

/* TRANSITION — płynne przejście przy zmianie stanu */
.btn {
  background: #7c3aed;
  transform: translateY(0);

  /* właściwość  czas  funkcja-czasowa  opóźnienie */
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1) 0s;
}
.btn:hover {
  background: #a855f7;
  transform: translateY(-4px);
  box-shadow: 0 0 20px rgba(124,58,237,0.4);
}

/* ANIMATION — pełna animacja z @keyframes */
.pulsing-dot {
  animation: pulse 2s ease-in-out infinite;
  /* nazwa  czas  funkcja  opóźnienie  powtórzenia  kierunek */
  animation: pulse 2s ease-in-out 0s infinite alternate;
}

@keyframes pulse {
  0%   { transform: scale(1);    opacity: 1; }
  50%  { transform: scale(1.1);  opacity: 0.7; }
  100% { transform: scale(1);    opacity: 1; }
}

@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50%      { transform: translateY(-16px); }
}

@keyframes fadeIn {
  from { opacity: 0; transform: translateY(24px); }
  to   { opacity: 1; transform: translateY(0); }
}

/* Funkcje czasowe */
/* ease          — wolno → szybko → wolno (domyślne) */
/* linear        — stała prędkość */
/* ease-in       — wolno na starcie */
/* ease-out      — wolno na końcu */
/* ease-in-out   — wolno na starcie i końcu */
/* cubic-bezier  — własna krzywa Béziera */

/* Użycie zmiennej przez var() */
.btn {
  background:    var(--kolor-glowny);
  padding:       var(--odstep-base);
  border-radius: var(--promien);
  box-shadow:    var(--cien);

Responsywność

Responsywność to zdolność strony do dostosowania wyglądu do różnych rozmiarów ekranu. Podejście mobile-first oznacza, że najpierw piszemy style dla mobile, a potem rozszerzamy dla większych ekranów.

/* BREAKPOINTS — standardowe punkty przełamania */
/* Mobile:  < 768px  (domyślne — mobile first) */
/* Tablet:  768px–1024px */
/* Desktop: > 1024px */

/* Styl bazowy (mobile) */
.grid {
  display: grid;
  grid-template-columns: 1fr; /* 1 kolumna na mobile */
  gap: 16px;
}

/* Tablet i większe */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr); /* 2 kolumny */
  }
}

/* Desktop i większe */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr); /* 3 kolumny */
  }
}

/* Inne przydatne media queries */
@media (max-width: 768px) {
  .sidebar { display: none; }
  .hamburger { display: flex; }
}

@media (prefers-reduced-motion: reduce) {
  * { animation: none !important; transition: none !important; }
}

@media (orientation: landscape) {
  .hero { min-height: 100vh; }
}

/* Responsywna typografia z clamp() */
h1 {
  /* min  preferowana  max */
  font-size: clamp(1.5rem, 5vw, 3rem);
}

/* Responsywny obraz */
img {
  max-width: 100%;
  height: auto;
}

/* Responsywny iframe (proporcje 16:9) */
.video-wrapper {
  position: relative;
  padding-bottom: 56.25%; /* 9/16 = 56.25% */
  height: 0;
  overflow: hidden;
}
.video-wrapper iframe {
  position: absolute;
  top: 0; left: 0;
  width: 100%; height: 100%;
}
